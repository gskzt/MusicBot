# GS KZT

Простой веб-сайт с системой авторизации и админ-панелью.

## Описание

Платформа GS KZT состоит из двух отдельных страниц:
- **Страница пользователей (index.html)** - форма ввода логина и пароля. При нажатии "Войти" данные сохраняются и отправляются в админ-панель
- **Админ-панель (admin.html)** - страница с авторизацией администратора. После входа админ видит все данные, которые вводили пользователи

## Установка и запуск

### Локальный запуск

1. Клонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в браузере для доступа к странице пользователей
3. Откройте `admin.html` в браузере для доступа к админ-панели

### Размещение на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы проекта в корень репозитория:
   - `index.html` - страница для пользователей
   - `admin.html` - страница админ-панели
   - `styles.css` - стили сайта
   - `user.js` - логика для страницы пользователей
   - `admin.js` - логика для админ-панели
   - `README.md` - документация
3. Перейдите в настройки репозитория (Settings)
4. В разделе "Pages" выберите источник (Source) - обычно "main" или "master" branch
5. Сохраните изменения
6. Ваш сайт будет доступен по адресу: `https://[ваш-username].github.io/[название-репозитория]`
